# 📬 Need Help?

## 🐞 Bug Reports

Found a bug?

Please create a **GitHub Issue** and include:

- Operating System
- Python Version
- Error Message
- Steps to Reproduce
- Expected Behavior
- Screenshots (if applicable)

---

## 💡 Feature Requests

Have an idea?

We love suggestions!

Please create a **Feature Request Issue** explaining:

- The problem you're trying to solve
- Your proposed solution
- Why it would improve VerityAgent

## ❓ Questions

If you have general questions about:

- AI Agents
